About UI Options

UI Options allows a user to customize a website to their own personal needs and preferences. It offers a variety of adaptations/customization including: text size, text style, line spacing, contrast themes, table of contents, enhance inputs and more.

For the site integrator, UI Options is another step in the accessibility and inclusivity of your website. Through UI Options' user driven customizations, a broader audience is able to access your content and materials.

See UI Options in action

UI Options is integrated here on the Floe Project site and can be access through the show preferences tab along the top of the page.

The UI Options Demo is where to try out the latest in development version.

Integrator Information

Obtain UI Options

UI Options is available as part of Fluid Infusion. You can obtain a copy of Infusion directly from GitHub or through NPM.

CMS Integrations

Plugins for some content management systems are available.

  • Wordpress
  • Drupal - maintained by OpenConcept

Guides and Documentation

If installing UI Options using NPM or from a release on GitHub, "Setting Up User Interface Options" will help installing and structuring your code.

Once UI Options is installed, "Working with UI Options" can help guide you through setting up your website's styles and markup to take advantage of UI Options.
