Security and Privacy Architecture

The Floe Project is working with the Global Public Inclusive Infrastructure to provide learners with friendly and comprehensive tools for sharing their data with the web sites, access features and assistive technologies that they trust. These tools will give learners complete control over which preferences are shared with any GPII-enabled services.

For more information and to see an example of the security architecture in action, please see the video below.

Floe is a project of the Inclusive Design Research Centre, OCAD University, funded by a grant from The William and Flora Hewlett Foundation.
