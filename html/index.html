floe flexible learning for open education

  • Paving the way toward inclusive Open Education Resources

    Floe is enabling inclusive access to personally relevant, engaging learning opportunities for the full diversity of learners and content producers.

    Through the Open Education Resources community, Floe makes tools that help transform, augment, and personalize the learning experience.

  • Learner Options Personalize interfaces and content to meet individual needs and preferences. Demo - Designs
  • Video Player An accessible, transformable, and open source HTML5 video player. Demo - Designs
  • Metadata Author Identify and help people find accessible resources. Demo - Designs
  • Inclusive Learning Design Handbook
    An Open Educational Resource (OER) designed to assist teachers, content creators, Web developers, and others in creating adaptable and personalizable resources.
    OER Authoring
    Designs for an accessible authoring environment.
  • FLOE Wiki
    Workspace for Floe design, development, and project planning.
    Floe on Github
    Github repository of all Floe source code.

Partnerships

OERPUB
OERPUB is using the Floe Video Player and Metadata Author to create rich, customizable learning content.
Raising the Floor
The Global Public Inclusive Infrastructure (GPII), a project of Raising the Floor, is building upon the foundation of Learner Options to create preference editing tools. This global project makes extensive use of Fluid Infusion.
OER Commons
OER Commons has integrated Learner Options across their entire site and has also integrated the Floe designs for content authoring in Open Author.
